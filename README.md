# 👁️ Doors Eyes

Стильный и быстрый юзербот для Telegram на базе **Pyrogram**, вдохновленный функционалом `@eyesseebot`. Работает прямо от имени твоего аккаунта в личных сообщениях.

## 🛠️ Функционал
* `.mute` — включает режим "Doors Eyes": автоматически ловит и удаляет любые новые входящие сообщения от собеседника в текущем ЛС.
* `.unmute` — отключает режим удаления для текущего чата.
* `.spam X текст` — запускает быстрый спам сообщениями (где `X` — количество, максимум 100). Автоматически стирает саму команду запуска.
* **Мониторинг изменений** — если собеседник отредактирует своё сообщение, бот тут же пришлёт в чат уведомление с текстом изменения.
